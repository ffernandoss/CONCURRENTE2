Bienvenido a la Simulación de Pelotas con ID - Distribución Exponencial

Esta es una simulación donde las pelotas caen y se almacenan en columnas formando una distribución exponencial.

Instrucciones

Utiliza los botones para detener o reiniciar la generación de datos. Observa cómo las pelotas caen y se almacenan en columnas.

Acerca de

Este proyecto fue desarrollado para demostrar la simulación de una distribución exponencial utilizando pelotas que caen y rebotan en pines.
